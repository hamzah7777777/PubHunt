# Alerts

> Dependencies: `colors.md`, `radius.md`

## Core Specs

- **Padding:** 20px
- **Radius:** 0px (`base`) — alerts are crisp rectangles
- **Border:** 2px solid (heavier than the default border to read as a notice block)
- **Heading:** 18px (VT323), normal weight
- **Body:** 16px (VT323), normal weight, 1.6 line-height

## Variants

- **Background:** brand-softer

- **Text:** fg-brand-strong
