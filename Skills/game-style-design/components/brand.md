# Brand

## Identity

- **Design system name:** Sega Design System
- **Project name:** PubHunt

## Logo Usage

- Treat the project logo as the primary brand mark for this design system.
- Use the logo in product headers, empty states, authentication screens, and branded dashboard surfaces when a brand cue is useful.
- Keep clear space around the logo; do not crop, distort, recolor, or place it on low-contrast surfaces.
- If no logo is set, use a simple text mark based on the design system or project name rather than inventing an unrelated symbol.

## Rules

- Pair logo usage with accessible text labels where the brand mark alone would be ambiguous.
- Use `colors.md`, `typography.md`, `layout.md`, and component modules for all visual values outside of logo usage.
